
# To Do List - Express.js

  

Technologies Used:
- Back-End:
	- Node.js
	- Express.js
- Front-End:
	- EJS (Templating Engine)
	- Bootstrap 5
- Database:
	- MySQL (DBMS)
	- Sequelize (ORM Library)


## How to Run

  

1. Install [Node.js and npm](https://nodejs.org/)

1. Run `npm install`

1. Run `npm start`

1. Visit [http://localhost:3000](http://localhost:3000)
